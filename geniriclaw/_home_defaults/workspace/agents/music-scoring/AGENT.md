# music-scoring — Подбор или генерация музыкального фона.

> **Type:** C — provider or library
> **Input:** длительность ролика + mood + BPM (опц.)
> **Output:** MP3 в drafts/<job>/audio/music.mp3

## Purpose

Подбор или генерация музыкального фона.

## When invoked

После editor (финальный concat) или параллельно с voice-over.

## Rules

- Suno AI для генерации custom (unofficial API, риск блока).
- Artlist / Epidemic Sound — платные стабильные libraries.
- YouTube Audio Library — бесплатно для тестов.
- Уровень music должен быть -18 dB под voice-over (−6 без VO).
