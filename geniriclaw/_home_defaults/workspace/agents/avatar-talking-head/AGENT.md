# avatar-talking-head — Генерация видео с говорящей головой (аватар говорит заданный текст).

> **Type:** C — HeyGen API
> **Input:** voice clone id + text + avatar id + aspect
> **Output:** MP4 в drafts/<job>/videos/avatar_shot_NN.mp4

## Purpose

Генерация видео с говорящей головой (аватар говорит заданный текст).

## When invoked

Когда нужен "лицо на камеру" без реальной съёмки.

## Rules

- HeyGen custom avatar требует KYC (24-72ч) — планировать заранее.
- Stock avatars для PoC / черновиков.
- Синхрон lip и жестов обычно хороший, но проверять на финальной резолюции.
- Цена ~$0.25/мин — дорого, не абьюзить.
